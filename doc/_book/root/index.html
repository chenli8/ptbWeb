介绍

当前版本 4.0.0

技术栈 React + React-Router + React-Redux + ES6 + Webpack + Gulp

使用说明

$ npm i                     安装所有依赖
$ npm run dev               开发:同时执行下面两个命令
$ npm run devBuild          开发:生成HTML
$ npm run devServer         开发:开启热替换
$ npm run p                 生产:打包
$ npm run pt                生产:打包 , 同时发布到远程 test , hera 上

webpack 配置说明

* webpack.base.conf.js      基本配置 entry
* webpack.dev.conf.js       开发配置 启动devServer
* webpack.devBuild.conf.js  开发配置 生成页面
* webpack.entry.conf.js     页面配置
* webpack.file.conf.js      文件配置 删除 拷贝 定义文件夹
* webpack.prod.conf.js      生产打包配置
